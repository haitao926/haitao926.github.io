商汤2020自动驾驶开发笔试题

商汤自动驾驶开发一共两个半小时，5个编程题。

3. 最短路径

假设自动驾驶测试场地只有东西走向的路以及南北走向的超级长的路，形成了一个巨大的网格；路口中间的道路都是1公里长，车辆可以再任意一个路口向四个方向行驶。

现在要为外界展示我们的成果，所以莹姐安排一场试乘体验活动，一共有N个记者参与了这次活动，他们分别站在不同路口，等待自动驾驶车接到他们回基地。自动驾驶车上已经做了1位安全员以及一位工程师，每次最多只能做两位记者；另外由于接到的记者扔到非基地外的路口是非常不礼貌的，所以每次接到的记者只能带回基地。

一直我们基地的坐标，以及这N位机制的坐标，你能帮莹姐算一下自动驾驶车最短的行驶里程吗？

输入描述

第一行输入两个整数X,Y，表示基地的坐标（-100<X,Y<100）,第二行输入一个整数N，表示记者的数量（0<=N<=24）之后的N行的第i行表示第i个记者的坐标，有两个整数xi和yi（-100<=X,Y<=100）

输出描述

一个整数D，表示自动驾驶车辆接完所有记者需要行驶的最短距离公里数。

示例1

1 1

3

4 3

3 4

0 0

输出示例

16
