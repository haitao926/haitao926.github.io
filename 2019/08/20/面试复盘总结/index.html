面试复盘总结

深兰科技 - ROS软件开发

技术面 + HR 面

技术面：可能是真的缺人，基本上没怎么问问题

  1. 自我介绍 + 项目介绍
  2. 室外导航的方式： GNSS 差分GPS
  3. move_base导航的配置问题
  4. 然后大部分是介绍目前他们团队的现状，主要工作，还交流了一些技术上的，大概意思是应用层开发没前途，嵌入式还是不错的。

总结：整个自我介绍与项目介绍还算流畅，项目介绍过程中没有充分体现有价值的技术细节。

希望能够明白的问题： 1. 团队情况 ；2 .组织架构 ；3. 所属职责； 4. 加班情况；5. 薪资福利。

HR面 （对HR印象很不好。）

  1. 没有充分准备HR可能问的问题。

影创科技-SLAM算法工程师

技术面：

  1. 笔试题，英文的数学题，相关的概率问题，不怎么会。
  2. 自我介绍 10min ，也就随便聊了聊，都是他不关心的。
  3. 说一些他关心的问题：自我驱动力与自我学习能力；数学不怎么好。
  4. 问了IMU相关你知道的，特征点你知道的，描述了ORB ，然后被发现了没做过项目。
  5. 要了大佬的微信，说好下次再来，我能不能下次再去决定了我是否能够走上SLAM的道路。

HR面：

  1. 薪资20W-50W，薪资组成；
  2. 加班，怎么看待加班问题；
  3. 上班距离，通勤时间。
  4. 手头有没有其他offer 。

总结：

这是一次愉快的挂面：很欣赏大佬的风格。注重自己两方面的能力：1。数学基础，学习能力，2. 做事的性格态度。

大佬指出的三个方向：IMU预积分； ORB_SLAM; 3d重建；准备从ORB_SLAM与3D重建入手。

这次感觉就是自己菜，要专注沉下心去学习，准备秋招。学习SLAM

地平线一面技术面

自我介绍+项目，问了几个简单问题。

  1. 简历填的不好，没有深度班吧，还是要重新写
  2. 面试官说我不合适，关注面试官的每一个问题，条理清晰
  3. 准备要问面试官的问题。

思科招聘会现场面试

获取免笔试资格，很惨痛的教训，每一场都要认真准备。要仔细了解置位信息，所负责的

  1. 自我介绍依然有问题，没能体现出自己的优点和长处。
  2. 简历依然要修改，把自己拿不准的没有把握的删掉，不给自己挖坑。
  3. 临场应变能力需要继续加强，保持警惕严厉的面试官不怕，温和的面试官不过分暴露
  4. 对于回答的问题，不仅仅局限于问题的回答，不是回答上和没回答上的问题。。
